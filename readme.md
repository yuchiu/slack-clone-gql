# Slack Clone

#### [Slack Clone Server Github Repository](https://github.com/yuchiu/Slack-Clone-Server)

## Usage

#### terminal A - Client

#### 1. install dependencies

```
npm install
```

#### 2. start application

```
npm start
```

The app will be running on http://localhost:8080

#### terminal B - Server

#### 1. install dependencies

```
npm install
```

#### 2. start application

```
npm start
```

The app will be running on http://localhost:8081
