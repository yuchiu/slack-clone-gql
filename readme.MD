# Usage

## Server - Terminal A

#### 0. cd into server directory

```
cd server
```

#### 1. install dependencies

```
npm install
```

#### 2. start server

```
npm start
```

The server will be listening on http://localhost:8081

## Client - Terminal B

#### 0. cd into client directory

```
cd client
```

#### 1. install dependencies

```
npm install
```

#### 2. start client

```
npm start
```
