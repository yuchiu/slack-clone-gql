# Slack Clone

## Usage

### Server - Terminal A

#### 1. install dependencies

```
npm install
```

#### 2. start application

```
npm start
```

Application will be running on http://localhost:8081

### Client - Terminal B

#### 1. install dependencies

```
npm install
```

#### 2. start application

```
npm start
```

Application will be running on http://localhost:8080
